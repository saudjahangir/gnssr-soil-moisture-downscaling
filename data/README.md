## Data Access

Raw datasets are not included in this repository due to size and licensing constraints.

Data sources used in this project:
- CYGNSS L3 Soil Moisture (NASA PO.DAAC)
- SMAP L3 Soil Moisture (NSIDC)
- Sentinel-1 GRD (Copernicus)
- Sentinel-2 L2A (Copernicus)
- SRTM DEM (NASA / USGS)

Users should download data directly from official repositories.
